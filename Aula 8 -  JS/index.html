Comando validos para JS

Por ID:
- document.getElementByID('id'); ou let elemento = document.getElementById('id');

Por Classe:
- document.getElementByClassName('classe'); ou let elemento = document.getElementsByClassName('classe');

Por Nome:
- document.getElementsByName('nome'); ou let elemento = document.getElementsByName('nome');

Por query selector:
- document.querySelector('#id');
- document.querySelector('.classe');
